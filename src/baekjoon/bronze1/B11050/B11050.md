# 백준 11050 - 이항 계수 1
## 문제 설명
자연수 N과 정수 K가 주어졌을 때, 이항 계수 N choose K를 구하는 문제이다.
---
## 문제 풀이 과정
파스칼의 삼각형을 활용한 DP로 풀었다. `C[i][0] = C[i][i] = 1` 로 초기화한 뒤, `C[i][j] = C[i-1][j-1] + C[i-1][j]` 점화식으로 채워나갔다. 처음에 배열 크기를 `N`으로 선언해 인덱스 초과 에러가 발생했고, `N+1`로 수정해 해결했다. 또한 초기화 루프 범위를 `i < n`으로 했다가 일부 케이스가 초기화되지 않는 문제가 있어 `i < N+1`로 수정했다.
