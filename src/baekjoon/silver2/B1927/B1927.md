# 백준 1927 - 최소 힙
## 문제 설명
정수가 주어졌을 때, 0이 입력되면 배열에서 가장 작은 값을 출력하고 제거하며, 그 외의 값은 배열에 추가하는 문제이다. 배열이 비어있을 때 0이 입력되면 0을 출력한다.
---
## 문제 풀이 과정
자바 라이브러리를 사용하지 않고 최소 힙을 직접 구현했다. `ArrayList`를 기반으로 `insert()`와 `deleteMin()` 메서드를 구현했으며, 삽입 시 `heapifyUp`, 삭제 시 `heapifyDown` 방식으로 힙 속성을 유지했다. 자식 노드가 하나만 있는 경우와 힙이 비어있는 경우에 대한 엣지 케이스 처리가 필요했다.
