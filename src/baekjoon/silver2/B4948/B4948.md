# 백준 4948 - 베르트랑 공준

## 문제 설명
자연수 n이 주어졌을 때, n보다 크고 2n보다 작거나 같은 소수의 개수를 출력하는 문제이다. 입력이 0이면 종료한다.

---

## 문제 풀이 과정

에라토스테네스의 체를 활용해 소수를 판별했다. `getPrimeNumbersCount(n, 2n)` 메서드를 만들어 범위 내 소수 개수를 반환하도록 구현했다.

처음에는 `for (int i = n; i <= m; i++)`로 n부터 탐색했는데, 문제 조건이 "n보다 크고"이므로 n을 포함하면 안 된다는 걸 놓쳤다. `n + 1`부터 탐색하도록 수정했다.

에라토스테네스의 체 내부 루프에서 `j++`로 작성해 모든 수를 false로 만드는 실수를 했다. 소수의 배수를 제거해야 하므로 `j += i`로 수정해야 했다. 이 부분이 핵심 로직이었는데 놓쳐서 전혀 엉뚱한 결과가 나왔다.

출력 시 `System.out.println(sb)`를 사용했는데, `sb`에 이미 `\n`이 포함되어 있어 마지막에 빈 줄이 추가됐다. `System.out.print(sb)`로 수정해 해결했다.
