# 프로그래머스 42576 - 완주하지 못한 선수

## 문제 설명
참가자 명단과 완주자 명단이 주어졌을 때, 완주하지 못한 선수의 이름을 찾는 문제이다.

---

## 문제 풀이 과정

처음에는 이중 반복문을 이용해 참가자와 완주자를 하나씩 비교하는 방식으로 접근했다. 하지만 시간복잡도가 커질 것 같아 다른 방법을 고민하게 되었고, 해시를 사용하는 방향으로 바꾸었다.

이후 `HashMap`을 사용해 참가자와 완주자의 이름을 각각 저장하고, 이름별 개수를 세는 방식으로 구현했다. `getOrDefault()`를 이용해 동일한 이름이 여러 번 등장하는 경우도 처리할 수 있도록 했다.

처음에는 두 개의 Map을 만들어 서로 비교하는 방식으로 코드를 작성했다. 없는 key를 처리하기 위해 `getOrDefault()`를 사용하고, 값이 다른 경우 해당 key를 정답으로 저장하도록 구현했다.

하지만 일부 테스트에서 계속 실패가 발생했다. 여러 케이스를 직접 만들어 확인해보던 중, `Integer` 값을 비교할 때 `!=` 연산자를 사용했던 부분이 문제라는 것을 알게 되었고, 이를 `equals()`로 수정한 뒤 정상적으로 통과할 수 있었다. 이 과정을 통해 자료구조를 사용하는 것뿐만 아니라, 비교 연산에서도 세부적인 부분을 정확히 처리해야 한다는 점을 확인할 수 있었다.
