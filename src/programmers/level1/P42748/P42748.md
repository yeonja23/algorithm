# 프로그래머스 42748 - K번째수

## 문제 설명
배열을 i번째부터 j번째까지 자른 후 정렬했을 때, k번째 수를 구하는 문제이다.

---

## 문제 풀이 과정

처음에는 `Arrays.copyOfRange()`로 배열을 자르고, `Arrays.sort()`로 정렬한 뒤 k번째 값을 꺼내는 방식으로 접근했다.

`copyOfRange()`를 사용할 때 from, to 인덱스를 잘못 설정해 `IllegalArgumentException: 4 > 2` 에러가 발생했다. from과 to의 순서를 바꿔서 넣은 것이 원인이었고, 직접 출력해보며 인덱스를 하나씩 확인해 수정했다. 또한 `copyOfRange()`의 to는 미포함이라는 점도 이 과정에서 다시 확인했다.

이후에는 `answer` 배열을 `{}`로 초기화한 채로 값을 넣으려 해서 인덱스 범위 초과 에러가 발생했다. `new int[commands.length]`로 크기를 명시적으로 초기화해야 한다는 것을 이번 문제를 통해 다시 한번 확인할 수 있었다. 배열을 선언할 때 크기를 명확히 지정하는 습관이 필요하다는 것을 느꼈다.
