# 프로그래머스 42746 - 가장 큰 수

## 문제 설명
숫자 배열이 주어졌을 때, 숫자들을 이어 붙여 만들 수 있는 가장 큰 수를 문자열로 return하는 문제이다.

---

## 문제 풀이 과정

`int[]` 배열을 `ArrayList<String>`으로 변환한 뒤, `Collections.sort()`에 `(a, b) -> (b + a).compareTo(a + b)` 커스텀 Comparator를 적용해 정렬했다. 단순히 첫 글자만 비교하면 `"3"`과 `"30"` 같은 케이스를 올바르게 처리할 수 없어, 두 문자열을 이어붙인 결과를 비교하는 방식을 사용했다.

모든 값이 `0`인 경우 결과가 `"000..."`이 되는 엣지 케이스를 처리하는 과정에서 String 비교에 `==` 대신 `equals()`를 사용해야 한다는 점을 다시 한번 확인했다.
